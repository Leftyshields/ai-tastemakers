# Tissue roles

## Implementation

- **id:** Implementation  
- **purpose:** Execute implementation work for the Build organ; coordinate cells that run molecules.  
- **inputs:** Handoff from Build (work package).  
- **outputs:** Handoff to cells; completion or failure report to Build.  
- **composition:** Contains cells (e.g. Worker).  
- **health:** Aggregates cell health; completion rules define when tissue is “done”.
