# Organ roles

## Build

- **id:** Build  
- **purpose:** Produce artifacts from Creator intent; own the implementation capability area.  
- **inputs:** Task or handoff from organism (e.g. “build artifact X”).  
- **outputs:** Handoff to tissues (e.g. Implementation) with work package.  
- **composition:** Contains tissues (e.g. Implementation).  
- **health:** Aggregates tissue health; reports to organism.
